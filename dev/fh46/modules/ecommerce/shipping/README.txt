DESCRIPTION:

This is a shipping API for ecommerce stores. In addition, this 
module has the ability to calculate shipping costs based on the 
total quantity of items in the shopping cart or the total cost of 
items. With each option, ranges (e.g., 1-5 items) can also be 
specified.
