Name: User Points
Author: Khalid Baheyeldin http://2bits.com and http://baheyeldin.com/khalid

Description:
------------
The separate User Points module now provides the ability to use points
as a method of payment integrated with the ecommerce package.

It also allows granting of points on every purchase (reward points).

Moreover, users can purchase points using real money.

Check the user points module for more detail at the following URL:
http://drupal.org/project/userpoints
