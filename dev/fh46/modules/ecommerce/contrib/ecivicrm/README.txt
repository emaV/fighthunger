DESCRIPTION:

This module integrates the Drupal E-Commerce module with CiviCRM.
It provides the following features:
 - logs donations and other transactions into the user's activity
   history. (donations are logged separately from other  
   financial transactions)

Upcoming functionality:
 - Utilize CiviCRM for address management, replacing the E-Commerce
   module's address.module.

********************************************************************
INSTALLATION:

Enable the ecivicrm module.
