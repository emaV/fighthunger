Name: Ecommerce Package Contributed Modules
********************************************************************
DESCRIPTION:

This is a collection of contributed modules used to drive an
ecommerce system from within Drupal.

********************************************************************
INSTALLATION:

Installation instructions are provided in each modules' directory.
Note that these modules are not part of the base ecommerce
installation, so you'll need to load the database tables if instructed
to do so.
